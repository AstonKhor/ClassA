How to start application:

## Step 1

In Terminal, run 
```
npm install
``` 
to install all relevent packages /n

## Step 2

In Terminal, run 
```
npm run db:init
```
to setup and populate database

## Step 3

In Terminal, run
```
npm start:dev
```
to open server

## Step 4

In different Terminal, run 
```
npm run build:dev
```
to compile with webpack and babel

Open browser to localhost:3000
